Установка

3 шага до первой записи

Билд пока не подписан Apple Developer ID — поэтому первый запуск через right-click. Это нормально для бета-версии.

1

Скачайте и установите

Откройте .dmg и перетащите Mymeet.ai в папку Applications.

2

Разблокируйте через System Settings

На macOS Sequoia при первом запуске вылезет «Apple could not verify...». Это нормально — мы пока без сертификата Apple. Подробная инструкция в FAQ ниже, в три клика.

3

Дайте разрешения

macOS попросит Screen Recording и Microphone — для захвата системного звука встречи. Разрешите оба.

FAQ

Если что-то пошло не так

«Apple could not verify Mymeet.ai is free of malware» — как открыть на Sequoia

На macOS 15 Sequoia Apple намеренно убрал старый right-click → Open для приложений без Apple Developer ID. Сейчас flow такой (один раз на Mac):

  1. В диалоге нажмите Done
  2. Откройте System Settings (Apple menu → System Settings)
  3. Выберите Privacy & Security в боковой панели
  4. Прокрутите вниз до раздела Security
  5. Увидите «Mymeet.ai was blocked because it is not from an identified developer» → нажмите Open Anyway
  6. Подтвердите Touch ID / паролем
  7. Двойной клик по Mymeet.ai в Applications → в новом диалоге нажмите Open
  8. Готово. Дальше открывается обычным двойным кликом.

Билд подписан ad-hoc и валиден (codesign проверяется), но не нотарифицирован Apple. Notarization требует Apple Developer ID ($99/год) — добавим в следующем релизе, тогда диалога не будет совсем.

«App is damaged and can't be opened» — старая ошибка

Если видите именно такое сообщение — скачайте свежий .dmg по кнопке выше: в первой версии была bug в подписи, в новой исправлено. Если ошибка повторяется на свежем билде: xattr -cr /Applications/Mymeet.ai.app && codesign --sign - --deep --force /Applications/Mymeet.ai.app в Terminal.

Не приходит уведомление о звонке

Mymeet распознаёт известные приложения по бандл-ID (Zoom, Teams, Gather, Slack, Discord, Webex, Контур.Толк, Telemost, СберДжазз). Если ваше приложение в этом списке нет — нажмите Cmd+Shift+R вручную.

Запись пустая после остановки

Скорее всего нет разрешения Screen Recording. Откройте System Settings → Privacy & Security → Screen Recording, включите тумблер для mymeet-audio-mac. Потом перезапустите приложение.

Какой нужен API-ключ?

На время бета-теста ключ выдаётся вручную команде sales@mymeet.ai. После релиза будет login через Google/Microsoft.

Большие встречи (>1 ГБ)

Бесплатный тариф mymeet.ai принимает файлы до 1 ГБ. Для встреч 2+ часов — нужен Pro тариф (до 3 ГБ).

Поддерживается только macOS?

Сейчас да, macOS 13+. Windows билд в roadmap, iOS/Android — ещё дальше.
