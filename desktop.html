Как пройти защитный экран Apple

Если macOS пишет, что не может проверить приложение, разрешите Mymeet.ai один раз в Privacy & Security. Без этого Desktop не откроется.

1. Разрешите запуск в Privacy & Security

Этот экран появляется после первой попытки открыть приложение из Applications.

  1. Откройте .dmg, перетащите Mymeet.ai в Applications и запустите приложение оттуда.
  2. Попробуйте открыть Mymeet.ai из Applications. Если появится предупреждение, нажмите Done.
  3. Откройте System Settings → Privacy & Security и пролистайте до блока Security.
  4. Нажмите Open Anyway рядом с Mymeet.ai. Apple показывает эту кнопку примерно час после первой попытки запуска.
  5. Подтвердите пароль или Touch ID, затем в новом диалоге нажмите Open.
Официальная инструкция Apple

2. Дайте разрешения для записи

После запуска macOS спросит доступы. Они нужны, чтобы Desktop записал обе стороны встречи.

Microphone Разрешите микрофон, чтобы в записи был ваш голос.
System Audio Разрешите запись системного звука, чтобы были слышны другие участники.
Restart Desktop Если macOS попросит перезапуск приложения, закройте Desktop и откройте его снова.

Минимум действий до готового саммари

Desktop нужен там, где бот во встрече не подходит: внешний звонок, приватный разговор, нативный Zoom или Teams.

Системный звук и микрофон

Записывает встречу и ваш голос, чтобы в транскрипте были обе стороны разговора.

Авто-детект звонка

Видит активный звонок и показывает компактное уведомление с кнопкой записи.

Загрузка в workspace

После остановки запись отправляется в app.mymeet.ai, где появляются транскрипт и саммари.

Локальная очередь

Если интернет пропал, запись остаётся на компьютере и догружается после восстановления связи.

На странице оставлен только путь, который реально нужен пользователю после установки.

FAQ

Если запуск или запись не сработали

Самые частые проблемы вынесены в короткие ответы. Сложные случаи лучше отправлять в поддержку с версией приложения.

Windows SmartScreen пишет Unknown publisher

Windows-версия пока может показывать предупреждение. Нажмите More info → Run anyway, затем продолжайте установку.

Не появилось уведомление о звонке

Запустите запись вручную из окна Desktop или tray-меню. Если приложение звонка редкое, пришлите его название в поддержку.

Запись пустая

Проверьте разрешения macOS или Windows на микрофон и системную запись аудио, затем перезапустите Desktop.

Большие встречи

Файл загружается в ваш Mymeet.ai workspace. Доступный размер зависит от тарифа и лимитов основного продукта.
